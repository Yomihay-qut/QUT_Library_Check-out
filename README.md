# 告别封号——QUT图书馆自动签退系统 

基于 Python 构建的图书馆自动预约签退工具，支持 Windows 系统一键部署后台计划任务，实现每天自动无感签退。已提供编译好的可执行文件，无需配置 Python 环境，零基础用户也能“开箱即用”。彻底告别封号！

**注意此项目需保证电脑开机才能运行，可以睡眠，请保证设备拥有校园网环境，后续考虑开发移动安卓版本。**

## ✨ 功能特性

1. **一键式部署**：双击即可申请管理员权限并向 Windows 系统注册隐藏计划任务。
2. **多账号管理**：支持同台电脑通过 `accounts.json` 配置多个账号，按序轮询读取执行。
3. **防固定时间特征检测**：内置任务触发时机的「随机时间抖动」（如设定 11:30，实际会在前后 10 分钟内浮动），避免账号触发异常风控。
4. **安装即时自检**：安装计划任务时会立即执行一次安全登录校验，若账号密码错误或网络不通导致自检失败，会自动回滚清理任务，避免产生无效进程。

## 快速上手 

1. 在仓库的 **Releases** 页面下载最新发布的 Windows 专版打包文件，解压到一个**后续不会随意移动或删除的文件夹**中（推荐放电脑 `D盘` 或文档目录）。
2. **双击运行** `install_task.exe`。此时程序会检测到无账号配置，并自动在同级目录下生成一个 `accounts.json` 模板文件，按回车退出即可。
3. 使用系统自带记事本打开 `accounts.json`，把里面的 `"your_account"` 替换为你的真实账号/学号，`"your_password"` 替换为密码。
    - *如需添加多人，只需复制花括号包裹的那块内容即可。*
4. **右键以管理员身份运行** `install_task.exe`（如果遇到杀毒软件报毒请放行，打包 exe 容易误报）。
5. 看到命令框提示 **`[安装成功] 任务已创建且即时验收通过`** 以及你的身份信息后，成功。

**说明：** 部署成功后，系统会在每天设定的时间（如 11:30、17:30、21:30 等窗口）在后台全静默自动完成签退。

### 🗑 卸载或停止运行
若要停止系统的自动签退，或者想要移动文件夹位置，直接运行目录内的 `uninstall_task.exe`，即可干净地擦除 Windows 系统底层的计划任务。

## 开发者指南 (源码运行)

如果你有 Python 环境并想自己修改或打包，可以按照以下步骤：

```bash
# 1. 克隆代码或下载源码
git clone https://github.com/yomihay/QUT_Library_Check-out.git
cd QUT_Library_Check-out

# 2. 安装依赖 (推荐使用虚拟环境)
pip install -r requirements.txt

# 3. 运行测试（仅仅执行一次完整的流程，不注册任务）
python main.py

# 4. 注册每天自动执行的计划任务
python install_task.py
```

### 自行打包命令
本项目提供了基于 PyInstaller 的 `.spec` 配置文件来处理 `Crypto` 复杂底层的依赖：
```bash
pip install pyinstaller
pyinstaller --clean -y install_task.spec
pyinstaller --clean -y main.spec
```
编译好的可执行文件将放置在 `dist/` 文件夹下。
